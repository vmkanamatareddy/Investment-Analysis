# Investment-Analysis-Exploratory Data Analysis
An Investment company  wants to invest in countries with the highest amount of funding for the chosen investment type. This is a part of its broader strategy to invest where most investments are occurring.
Firm wants to see the top nine countries which have received the highest total funding (across ALL sectors for the chosen investment type)
For the chosen investment type, make a data frame named top9 with the top nine countries (based on the total investment amount each country has received)
Identify the top three English-speaking countries in the data frame top9.
